# Toronto-Bike-Share-2024

## Introduction
An SQL analysis of Bike Share Ridership Data in Toronto between January and September 2024, alongside weather information, to explore patterns in usage, peak hours, membership types, and route popularity.

## Project Overview
The goal of this project is to understand: 
- How weekday VS weekend usage and duration differed
- How casual members' and annual members' usage differed
- How weather impact bike share ridership
- Peak Hours
- Popular Route

This analysis demonstrates SQL data manipulation, aggregation, window functions, and basic data cleaning.
For a full step-by-step analysis and SQL queries, see [ANALYSIS.md] (ANALYSIS.md)
