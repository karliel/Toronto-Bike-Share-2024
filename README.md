# Toronto-Bike-Share-2024

**Project Overview**
An SQL analysis of the usage of Bike Share in Toronto between January and September 2024

**Datasets**
**Toronto Bike Share Ridership Data 2024**
https://open.toronto.ca/dataset/bike-share-toronto-ridership-data/

**Sampled Toronto Bike Share Ridership Data 2024**

**Toronto Daily Weather Data Report 2024**
en_climate_daily_ON_6158731_2024_P1D.csv

Due to GitHub file size limits, raw datasets for bike share riderships are not included. A cleaned sampled subset of the datasets is provided for reference. The full analysis was performed on the complete data.
